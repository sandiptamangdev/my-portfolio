The Story So Far
I'm Sandip Tamang, a developer and designer based in Kathmandu, Nepal. My journey started with curiosity about how things work, which turned into building tools that blend clean code with thoughtful design—from color palette generators that understand visual harmony to AI assistants that actually reason.
Right now, I'm focused on creating experiences that feel natural and work beautifully. I prototype fast, debug obsessively, and refactor until it's right. Currently, I'm deep into AI research with LLAI, exploring how machines can maintain context and think more like humans—it's still in the lab, but that's where the interesting stuff happens.
My toolkit has grown to include Python, JavaScript, HTML/CSS, Tailwind, and Node.js, but the real skill is the process: break things, learn from every bug, and ship something better. I'm constantly iterating on my projects—check my GitHub and you'll see something's always changing. Whether it's perfecting smooth animations or testing color algorithms against accessibility standards, I'm learning that tiny improvements compound into something meaningful. This is just the beginning.
